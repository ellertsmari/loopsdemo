for loop: one sentence

A for loop packs the three questions into one line: start the counter, keep looping while the condition is true, and move the counter each time.

Start the counter
Check the condition
Move forward

for (let i = 1; i <= 5; i = i + 1) {
  console.log("Looping is learning", i);
}

while loop: check first

A while loop separates the parts: set up before the loop, check at the top, repeat the body, then move the counter inside.

Edit the highlighted lines to mirror the story and see how the logic stays the same.

let i = 1;
while (i <= 5) {
  console.log("Looping is learning", i);
  i = i + 1;
}

forEach: trust the array

Array.prototype.forEach handles the counting for you. Choose the array, then describe what should happen for every item.

Edit the array or the callback to see different sequences.

const values = [2, 4, 6];
values.forEach((value, index) => {
  console.log("Value", index, value);
  // forEach moves to the next item automatically
});

Try these mini challenges

Say the loop sentence out loud before you look at the answer. Focus on the three decisions each time.

Print the numbers 1 through 4 on separate lines.
for (let i = 1; i <= 4; i = i + 1) { console.log(i); }
Count down from 5 to 1 and then show "Blast off!".
for (let i = 5; i >= 1; i = i - 1) { console.log(i); } console.log("Blast off!");
Add up the first 3 multiples of 5 (5, 10, 15).
let total = 0; for (let i = 1; i <= 3; i = i + 1) { total = total + 5 * i; } console.log("Total:", total);
Once the three questions are clear, every loop becomes a small variation. Keep rewriting the sentence for your own exercises.
